# Student Assistant Chatbot

This is a full-stack AI-powered chatbot designed for Indian students (up to 10th standard).

## Features
- Subject-specific windows (Maths, Science, Social, Computers)
- AI explanations with step-by-step clarity
- Supports OpenAI, Groq, and Perplexity APIs
- React frontend + Node.js backend

## Setup

### Backend
```bash
cd backend
npm install
npm run dev
```

### Frontend
```bash
cd frontend
npm install
npm run dev
```

### API Keys
Copy `.env.example` to `.env` and set your API keys.
